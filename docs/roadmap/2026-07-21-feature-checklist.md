# 위키(Confluence 클론) 기능 요구사항 체크리스트

> 살아있는 개발 트래커. 개발하면서 `- [ ]` → `- [x]`로 체크한다.
> 최초 작성 2026-07-21 (사용자 제공 요구사항 원본). 기존 `docs/roadmap/2026-07-17-platform-roadmap.md`(전역 결정·저장 포맷 플래그)와 함께 본다.
>
> **상태 표기 규약**
> - `- [ ]` 미착수 / 미확인
> - `- [x]` 완료(코드로 검증됨)
> - `⚠️ 확인필요` = 구현돼 있을 수 있으나 실제 동작 미검증 — 개발 착수 전 코드로 확인
> - `❓ 정책대기` = 백엔드/제품 정책 결정 필요 (`docs/backend/` 정책 대기 목록과 연동)

## 1. 스페이스

- [ ] 스페이스 목록 조회
- [ ] 스페이스 생성 (이름 + 키)
- [ ] 키 대문자 자동 변환
- [ ] 스페이스 즐겨찾기 ⚠️ 확인필요 (별표 UI는 있는데 실제 동작?)
- [ ] 스페이스 설정 (이름/설명 수정)
- [ ] 스페이스 삭제/보관(archive) ❓ 정책대기 (삭제 vs 보관 구분)
- [ ] 스페이스 홈 페이지 (스페이스 개요 + 최근 업데이트)
- [ ] 스페이스 아이콘/색상 커스텀
- [ ] 스페이스별 권한 (멤버, 읽기/쓰기 구분) ❓ 정책대기

## 2. 페이지 CRUD & 구조

- [ ] 페이지 생성
- [ ] 페이지 조회 (마크다운 렌더링)
- [ ] 페이지 편집
- [ ] 페이지 삭제
- [ ] 트리 구조 (부모-자식 계층)
- [ ] 하위 페이지 목록 표시
- [ ] 페이지 이동 (다른 부모/다른 스페이스로)
- [ ] 트리에서 드래그앤드롭으로 순서/계층 변경
- [ ] 페이지 복제
- [ ] 삭제 시 휴지통 (soft delete + 복원) ❓ 정책대기
- [ ] 하위 페이지가 있는 페이지 삭제 시 처리 정책 (같이 삭제? 승격?) ❓ 정책대기

## 3. 에디터

- [ ] 리치 텍스트 툴바 (굵게, 기울임, 취소선, 코드 등)
- [ ] 블록 타입: 제목 1~3, 글머리/번호/체크박스 목록, 인용
- [ ] 패널 (노트/팁/경고/주의) ⚠️ 저장 포맷 재논의 (마크다운 비표현 블록 — 로드맵 플래그 확인)
- [ ] 코드 블록 (구문 강조)
- [ ] 표
- [ ] 구분선, 링크, 이미지(URL), 이모지
- [ ] 슬래시(/) 커맨드 메뉴
- [ ] 이미지 파일 업로드 (URL 말고 드래그앤드롭/붙여넣기) ❓ 정책대기 (스토리지)
- [ ] 표 편집 UX (행/열 추가·삭제, 헤더 토글)
- [ ] 내부 페이지 링크 (@페이지명 또는 [[ ]] 자동완성)
- [ ] 임시 저장 (draft) / 자동 저장
- [ ] 편집 중 이탈 경고 (저장 안 된 변경사항)
- [ ] 목차 자동 생성 매크로
- [ ] 첨부파일 업로드 ❓ 정책대기 (스토리지)

## 4. 버전 관리

- [ ] 히스토리 버튼 ⚠️ 확인필요 (동작 여부?)
- [ ] 버전 목록 (버전 번호, 수정자, 일시)
- [ ] 버전 간 diff 보기
- [ ] 특정 버전으로 복원
- [ ] 수정 시 "변경 요약" 코멘트 입력

## 5. 검색

- [ ] 사이드바 페이지 검색 (제목 기준)
- [ ] 본문 전문(full-text) 검색 ❓ 정책대기 (검색 인프라)
- [ ] 전역 검색 (모든 스페이스 대상, 상단 검색바)
- [ ] 검색 결과 하이라이팅
- [ ] 최근 본 페이지

## 6. 사용자 & 권한

- [ ] 로그인/로그아웃
- [ ] 사용자 표시 (이름, 아바타)
- [ ] 회원가입 / 사용자 관리
- [ ] 프로필 페이지
- [ ] 권한 체계 (관리자/멤버/읽기 전용) ❓ 정책대기
- [ ] 페이지 제한 (특정 페이지 비공개) ❓ 정책대기

## 7. 협업

- [ ] 댓글 (페이지 하단)
- [ ] 인라인 댓글 (본문 특정 텍스트에)
- [ ] 멘션 (@사용자)
- [ ] 좋아요/리액션
- [ ] 페이지 watch (변경 알림)
- [ ] 알림 (내가 멘션됨, watch 중인 페이지 수정됨)
- [ ] 동시 편집 충돌 처리 ❓ 정책대기 (최소: 편집 잠금/충돌 감지, 최대: 실시간 공동 편집)

## 8. 부가 기능

- [ ] 다크 모드 토글 ⚠️ 확인필요 (실제 동작?)
- [ ] 전체 너비 토글
- [ ] 페이지 공유 링크 복사
- [ ] 페이지 내보내기 (PDF/마크다운)
- [ ] 템플릿 (회의록, 기획서 등 미리 정의된 페이지 양식)
- [ ] 라벨/태그
- [ ] 최근 업데이트 피드 (대시보드)
- [ ] 키보드 단축키 (e = 편집, / = 검색 등)

- 히스토리 버튼 동작
- 다크 모드 토글 동작
